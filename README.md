Repeat a string one or more times:

```
repeats(string[, number])
```

#### Example

```js
var repeats = require('repeats');
console.log(repeats('na', 16), 'BATMAN!');
// nananananananananananananananana BATMAN!
```

## Installation

```sh
$ npm install repeats
```

## Testing

```sh
$ npm test
$ npm run lint
```
